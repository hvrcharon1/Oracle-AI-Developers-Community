# Visual Studio Code (VS Code) Integration with Oracle AI

Visual Studio Code is the premier IDE for modern Oracle developers, offering deep integration with Oracle AI Database and OCI through specialized extensions.

## Oracle SQL Developer for VS Code

The **Oracle SQL Developer for VS Code** extension is the cornerstone of the Oracle development experience in VS Code. It brings industry-leading SQL and PL/SQL development capabilities directly into the editor [1].

### AI-Powered Features:
*   **Natural Language to SQL**: Democratizes data access by allowing users to ask questions in plain English. The extension's AI agents generate the corresponding SQL queries, eliminating the need for deep SQL knowledge [1].
*   **MCP Server Integration**: Seamlessly connects with the **Oracle SQLcl MCP Server**. This allows AI agents to query the database's data dictionary to discover accurate object names and execute SQL directly within the IDE [2].
*   **Schema Annotation**: Developers can annotate database schemas directly in the database. These annotations help AI agents understand complex or poorly documented schemas, leading to more accurate query generation [1].
*   **26ai Support**: Includes specialized tools for managing **JSON Relational Duality Views** in Oracle Database 26ai [1].

## Oracle Developer Tools for VS Code (SQL and PL/SQL)

This extension provides essential tools for managing Oracle databases on OCI, including:
*   **OCI Integration**: Start, stop, and terminate OCI database instances directly from VS Code.
*   **AI Profile Configuration**: Easily configure AI profiles to leverage OCI Generative AI services within your development workflow [3].
