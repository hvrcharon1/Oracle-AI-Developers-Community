# Oracle SQL Developer Integration with AI

Oracle SQL Developer has long been the standard for Oracle database development. It is now evolving to include deep AI and OCI integration.

## SQL Developer Next (VS Code Extension)

The future of SQL Developer is the **SQL Developer for VS Code** extension. This is where Oracle is focusing its AI innovation, including:
*   **Natural Language to SQL**: Translating plain English into complex SQL queries.
*   **MCP Server Support**: Enabling AI agents to interact directly with the database.

## SQL Developer Web (OCI Console)

Oracle SQL Developer Web is now integrated directly into the **OCI Console**. This provides a browser-based environment for:
*   **SQL Worksheet**: Executing queries and scripts on cloud databases.
*   **Data Modeler**: Designing and visualizing database schemas.
*   **SQL Monitoring**: Real-time monitoring of SQL execution on OCI [1].

## Classic SQL Developer

The classic desktop version of SQL Developer continues to provide robust support for:
*   **OCI Connection Management**: Easily connecting to Autonomous Databases and other OCI-hosted instances.
*   **Cloud Database Migration**: Tools for migrating on-premises databases to OCI.
